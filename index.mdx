    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Etsuko in just a few steps
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Etsuko repository from GitHub to your local machine.
      
      ```bash
      git clone https://github.com/lazpeng/etsuko.git
      cd etsuko
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install the required dependencies for your platform. On Linux or macOS with a package manager:
      
      ```bash
      # Example for Ubuntu/Debian
      sudo apt-get install cmake ninja-build libglew-dev libglfw3-dev libopenal-dev
      ```
      
      <Info>See the [Requirements](/building/requirements) page for detailed platform-specific instructions.</Info>
    </Step>
    
    <Step title="Build and run">
      Configure and build the project using CMake with the desktop-release preset.
      
      ```bash
      cmake --preset desktop-release
      cmake --build --preset desktop-release
      cd ./build/desktop-release
      ./etsuko
      ```
      
      <Note>You'll need to configure a song file in `config.c` before running. See the [Song Format Guide](/song-format/overview) for details.</Note>
    </Step>
  </Steps>
</div>
