## Overview

The renderer system (`renderer.h`) provides a hardware-accelerated OpenGL rendering pipeline with support for textures, text, shadows, blur effects, and compositing through render targets.

<Info>
  All rendering functions should be called **only from the main thread**. No synchronization is in place for concurrent access.
</Info>

## Initialization

```c
void render_init(void);
void render_finish(void);
```

**`render_init()`** initializes:
- Window creation (desktop) or canvas setup (web)
- OpenGL context and state
- Shader programs
- Viewport and projection matrices

**`render_finish()`** cleans up all renderer resources. Should only be called once at application shutdown.

<Note>
  `render_init()` will abort on error. If it returns, initialization succeeded.
</Note>

## Texture System

### Texture Structure

```c
typedef struct Texture_t {
    unsigned int id;           // OpenGL texture id
    int32_t width, height;     // Dimensions in pixels
    float border_radius;       // Optional border radius for rendering
    unsigned int vbo, vao;     // Pre-configured vertex objects
    int32_t buf_x, buf_y, buf_w, buf_h;  // Cached buffer configuration
} Texture_t;
```

Textures include:
- **OpenGL texture ID**: Reference to GPU texture
- **Dimensions**: Cached width/height
- **Border radius**: For rounded rectangle rendering
- **Vertex objects**: Pre-configured VAO/VBO for performance
- **Buffer cache**: Tracks when vertex objects need reconfiguration

### Creating Textures

<Accordion title="Text Textures">
  ```c
  Texture_t *render_make_text(
      const char *text,
      int32_t pixels_size,
      const Color_t *color,
      FontType_t font_type
  );
  ```
  
  Creates a texture from a text string using TrueType font rendering.
  
  **Parameters**:
  - `text`: UTF-8 encoded string
  - `pixels_size`: Font size in pixels
  - `color`: Text color (RGBA)
  - `font_type`: `FONT_UI` or `FONT_LYRICS`
  
  **Returns**: Texture sized exactly to contain the rendered text
  
  <Note>
    No centering, alignment, or wrapping is performed. The UI system handles layout.
  </Note>
</Accordion>

<Accordion title="Image Textures">
  ```c
  Texture_t *render_make_image(
      const unsigned char *bytes,
      int length,
      double border_radius_em
  );
  ```
  
  Creates a texture from raw image data.
  
  **Supported formats**: JPEG, PNG
  
  **Parameters**:
  - `bytes`: Raw image file data
  - `length`: Data length in bytes
  - `border_radius_em`: Border radius in em units (optional)
</Accordion>

<Accordion title="Null Textures">
  ```c
  Texture_t *render_make_null(void);
  ```
  
  Creates a null texture that's a no-op when drawn. Useful for:
  - Dynamic drawables that can't be cached
  - Placeholder for custom rendering
  - Pre-allocating vertex objects for rounded rectangles
</Accordion>

### Destroying Textures

```c
void render_destroy_texture(Texture_t *texture);
```

Frees all resources associated with a texture. Does not affect shadows created from it (must be freed separately).

## Drawing Operations

### Drawing Textures

```c
void render_draw_texture(
    Texture_t *texture,
    const Bounds_t *at,
    const DrawTextureOpts_t *opts
);
```

**Core drawing function**. Renders a texture to the current render target (framebuffer or render target).

**Bounds Definition**:
```c
typedef struct Bounds_t {
    double x, y;        // Position (relative to screen or parent)
    double w, h;        // Dimensions
    double scale_mod;   // Scale modification (0 = normal, >0 = larger, <0 = smaller)
} Bounds_t;
```

<Info>
  **Scale modifier**: `final_scale = 1.0 + scale_mod`
  - `0.0` = normal size
  - `0.5` = 1.5x size (50% larger)
  - `-0.25` = 0.75x size (25% smaller)
  - Lower bound: `-1.0` (zero size)
</Info>

**Draw Options**:
```c
typedef struct DrawTextureOpts_t {
    int32_t alpha_mod;                         // Alpha (0-255, 255 = opaque)
    float color_mod;                           // Color multiplier (1.0 = original)
    const DrawRegionOptSet_t *draw_regions;    // Partial texture visibility
    const ScaleRegionOptSet_t *scale_regions;  // Per-region scaling
    bool center_on_scale;                      // Center when scaled
} DrawTextureOpts_t;
```

### Drawing Rounded Rectangles

```c
void render_draw_rounded_rect(
    const Texture_t *null_tex,
    const Bounds_t *bounds,
    const Color_t *color,
    float border_radius
);
```

Draws a solid rounded rectangle. Requires a null texture for vertex object caching.

```c
#define BORDER_RADIUS_AUTO (-1.f)
```

Use `BORDER_RADIUS_AUTO` to automatically calculate radius as `min(width, height) / 2`.

## Shadow System

### Shadow Structure

```c
typedef struct Shadow_t {
    Texture_t *texture;    // Shadow texture (larger than source)
    Bounds_t bounds;       // Shadow dimensions
    int32_t offset;        // Offset from source (x and y)
} Shadow_t;
```

### Creating Shadows

```c
Shadow_t *render_make_shadow(
    Texture_t *texture,
    const Bounds_t *src_bounds,
    float blur_radius,
    int32_t offset
);
```

Generates a shadow texture from a source texture:

1. Creates a black silhouette of the source
2. Applies Gaussian blur
3. Erases the area where the source will sit (based on offset)
4. Adds padding for blur

<Note>
  Shadow textures are always **larger** than the source due to blur padding. Draw shadows before the source texture for correct layering.
</Note>

### Drawing Shadows

```c
Shadow_t *shadow = render_make_shadow(texture, &bounds, 10.0f, 8);

// Draw shadow first (apply offset manually)
Bounds_t shadow_bounds = shadow->bounds;
shadow_bounds.x = bounds.x + shadow->offset;
shadow_bounds.y = bounds.y + shadow->offset;
render_draw_texture(shadow->texture, &shadow_bounds, &opts);

// Draw source texture on top
render_draw_texture(texture, &bounds, &opts);
```

### Destroying Shadows

```c
void render_destroy_shadow(Shadow_t *shadow);
```

Frees shadow resources. Does not affect the source texture.

## Render Targets

Render targets enable texture compositing by rendering to textures instead of the framebuffer.

### Render Target Structure

```c
typedef struct RenderTarget_t {
    Texture_t *texture;                // Resulting texture
    struct RenderTarget_t *prev_target; // Previous target (for stacking)
    unsigned int fbo;                   // Framebuffer object
    int viewport[4];                    // Viewport configuration
    float projection[16];               // Projection matrix
} RenderTarget_t;
```

### Creating Render Targets

```c
const RenderTarget_t *render_make_texture_target(int32_t width, int32_t height);
Texture_t *render_restore_texture_target(void);
```

**Workflow**:

```c
// Create render target
const RenderTarget_t *target = render_make_texture_target(512, 512);

// All draw calls now render to the target
render_draw_texture(texture1, &bounds1, &opts1);
render_draw_texture(texture2, &bounds2, &opts2);

// Restore previous target and get resulting texture
Texture_t *result = render_restore_texture_target();

// Draw the composite texture
render_draw_texture(result, &final_bounds, &opts);

// Clean up
render_destroy_texture(result);
```

<Info>
  Render targets can **stack**. Creating a new target while one is active saves the previous target, which is automatically restored when calling `render_restore_texture_target()`.
</Info>

## Blur Effects

```c
Texture_t *render_blur_texture(const Texture_t *source, float blur_radius);

```

**`render_blur_texture`**: Creates a new blurred texture, preserving the source.

## Draw Regions

Draw regions allow rendering portions of a texture.

```c
typedef struct DrawRegionOpt_t {
    float x0_perc, x1_perc;  // X range (0.0 to 1.0)
    float y0_perc, y1_perc;  // Y range (0.0 to 1.0)
} DrawRegionOpt_t;

typedef struct DrawRegionOptSet_t {
    DrawRegionOpt_t regions[MAX_DRAW_SUB_REGIONS];
    int32_t num_regions;
} DrawRegionOptSet_t;
```

Example: Draw bottom-left quarter:
```c
DrawRegionOptSet_t regions = {
    .regions = {{
        .x0_perc = 0.0f, .x1_perc = 0.5f,
        .y0_perc = 0.0f, .y1_perc = 0.5f
    }},
    .num_regions = 1
};
```

## Scale Regions

Scale regions apply per-region scaling in the geometry shader.

```c
typedef struct ScaleRegionOpt_t {
    float x0_perc, x1_perc, y0_perc, y1_perc;
    float from_scale, to_scale;  // For animations
    float relative_scale;         // Scale modifier
} ScaleRegionOpt_t;

typedef struct ScaleRegionOptSet_t {
    ScaleRegionOpt_t regions[MAX_SCALE_SUB_REGIONS];
    int32_t num_regions;
} ScaleRegionOptSet_t;
```

<Info>
  `relative_scale` works like `Bounds_t.scale_mod`: `0.0` = normal, positive = larger, negative = smaller.
</Info>
